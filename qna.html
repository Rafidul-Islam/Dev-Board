Question-2 : What is the difference between innerHTML, innerText, and textContent ?

innerText : This selects the inner written text from the element and returns only the text.
Example : <h1 id="heading">Hello, World!</h1>
document.getElementById("heading").innerText = Hello, World! [This returns only the visible text]

innerHTML : This selects the full element includin the text and tagname.
Example : <h1 id="heading">Hello, World!</h1>
document.getElementById("heading").innerHTML = <h1 id="heading">Hello, World!</h1> [This returns the full Element]

textContent :Didnt know it earlier,had to google it.This Shows the hidden text in an alert box.
Example : <h1 id="heading" class="hidden">Hello, World!</h1>
document.getElementById("heading").textContent =alert( "Hello, World!") [This returns the text and shows in an alert box]

Question-3 : What is event delegation in the DOM?

Event delegation is a process where you can manipulate the task of the parent node one time istead of creating event for multiple child under the parent.This reduces the hastle of implementing multiple events in the document.
We mainly use event delegation to
  1.Save memory
  2.Improve performance and
  3.Clean Code.

Question-4 : What is event bubbling in the DOM?

Event bubbling is when you click in the page element, it starts the event from the target element and goes upward till its root element.
Like if you click in list items it will go to its list parent(ul/ol),then go to ol's parent ,then go to ol's parent's parent.This will conntinue until it reaches the root document.

Question-5: How do you create, add, and remove elements using JavaScript?

STEP 1: Get the element where you want to create a new element.
Example :
const parent = document.getElementById("fruits");

STEP 2: Create a new element using createElement('p').
Example :
const p = document.createElement('p');

STEP 3: Set value,text and attributes to the element.
Example :
p.innerText = "Hello World !";

STEP 4: Append The child to the parent Element
Example :
parent.appendChild(p);

STEP 5: To remove Child use removeChild() function;
Example :
parent.removeChild(p);
